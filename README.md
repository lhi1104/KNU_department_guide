#경북대학교 단과대학 설명#

##인문대학##
[경북대인문대학사진](https://www.veritas-a.com/news/articleView.html?idxno=602147)
경북대학교 인문대학은 1951년 경북대학교 문리과대학을 시작으로 1982년 현재의 인문대학을 자리 잡았다.
인문대학은 문학 사학, 철학을 중심으로 11개 학과의 학부와 대학원, 4개의 연구소로 구성되어있다. 
인문대학에 속한 학과는 고고인류학과, 국어국문학과, 노어노문학과, 독어독문학과, 불어불문학과, 사학과, 영어영문학과, 일어일문학과, 중어중문학과, 철학과, 한문학과가 있다.
