# 경북대학교 사범대학
**Teachers College, Kyungpook National University**
## 경북대학교 사범대학 소개
경북대학교 사범대학은 우리나라 중등교사 양성을 목적으로 광복 이듬해 개교하여 반세기 넘게 중등교육 발전에 실질적으로 지대한 공헌을 함은 물론 국내외의 대학과 연구소에서 중요한 연구를 수행하는 전문 학자를 많이 배출하였으며, 현대적인 교육시설을 갖추고 다양한 지원과 풍부한 시설을 이용하여 수준 높은 교육과 연구의 기회를 제공하고 있는 교원양성기관이다.

## 비전
**70여년 역사에 빛나는 중등 교원의 산실, 미래 사회를 이끌어 갈 사범교육의 중심, 세계로 비상하는 인재들의 요람**
- 교육적 가치 준수
- 효율적인 전문성 지향
- 사회변화의 혁신 선도
- 지정의가 조화를 이룬 전인 양성
- 인간발달 능력 극대화
