# 경북대학교 의과대학 소개
경북대학교 의과대학은 대한민국 대구에 위치한 국립 의과대학으로, 1923년에 설립된 오랜 역사를 가진 교육기관이다. 지역 의료의 중심 역할을 수행하며, 교육-연구-임상 진료를 균형있게 수행하고 있다.

## 1. 교육 과정

#### 🔹 의예과 (2년) — 기초 역량 형성 단계

| 분야 | 주요 과목 |
|------|-----------|
| 기초과학 | 생명과학, 화학, 물리학 |
| 의학 입문 | 의학입문, 의학통계학, 의학행동과학 |
| 윤리 · 사회 | 의사와 윤리, 환자-의사-사회 |
| 교양 | 의학의 역사 |

> 단순한 지식 습득을 넘어, 의료인으로서의 기본 태도와 사고방식을 형성하는 단계입니다.

<br>

#### 🔹 의학과 (4년) — 임상 역량 심화 단계

```
1~2학년  │  기초의학  │  해부학, 생리학, 생화학, 병리학 등
3~4학년  │  임상의학  │  내과, 외과, 소아과, 산부인과 등
         │  임상실습  │  경북대학교병원 현장 실습
```

<br>

---
## 2. 부속병원
경북대학교 병원과 연계하여 임상 실습 및 진료 경험을 제공한다.
