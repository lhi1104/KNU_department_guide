# 경북대학교 의과대학 소개
경북대학교 의과대학은 대한민국 대구에 위치한 국립 의과대학으로, 1923년에 설립된 오랜 역사를 가진 교육기관이다. 지역 의료의 중심 역할을 수행하며, 교육-연구-임상 진료를 균형있게 수행하고 있다.

## 1. 교육 과정

#### 🔹 의예과 (2년) — 기초 역량 형성 단계

| 분야 | 주요 과목 |
|------|-----------|
| 기초과학 | 생명과학, 화학, 물리학 |
| 의학 입문 | 의학입문, 의학통계학, 의학행동과학, 의대생 세미나 |
| 윤리 · 사회 | 의사와 윤리, 환자-의사-사회 |
| 교양 | 의학의 역사, 대학글쓰기 |

> 단순한 지식 습득을 넘어, 의료인으로서의 기본 태도와 사고방식을 형성하는 단계입니다.

<br>

#### 🔹 의학과 (4년) — 임상 역량 심화 단계

```
1~2학년  │  기초의학  │  해부학, 생리학, 생화학, 병리학 등
3~4학년  │  임상의학  │  내과, 외과, 소아과, 산부인과 등
         │  임상실습  │  경북대학교병원 현장 실습
```

<br>

---
## 2. 부속병원
경북대학교 병원과 연계하여 임상 실습 및 진료 경험을 제공한다.

### 3. 🔬 연구 분야

경북대학교 의과대학은 다양한 최신 의학 연구를 수행하고 있습니다.

- 🧬 종양학 (암 연구)
- 🛡️ 면역학
- 🧠 신경과학
- ❤️ 심혈관 질환
- ⚗️ 분자의학 및 의공학

> **학생 교실원 제도** 및 **의사과학자 양성 프로그램**을 통해  
> 임상과 연구 역량을 동시에 갖춘 인재를 양성합니다.

<br>

### 4. 🎯 교육철학 & 인재상

```
경북대학교 의과대학이 추구하는 인재

  ✅ 윤리적 판단 능력
  ✅ 환자와의 소통 능력  
  ✅ 사회적 책임 의식
  ✅ 문제 해결 능력
```

의예과 단계부터 **의료윤리**와 **사회적 맥락**을 교육에 포함시켜,  
환자 중심의 사고를 갖춘 의료인을 양성하는 것을 목표로 합니다.
